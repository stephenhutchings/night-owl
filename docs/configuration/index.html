Configuration

Add a file called nightowl.js to the root directory of your project to change the default configuration. Configuration files are optional.

The file is a module that declares an object as the default export. A basic configuration file might set input and output directories, like this:

export default {
  // The location of your source files
  src: "./src",

  // The output destination for built files
  dist: "./dist",
}
